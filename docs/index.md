# AetherLink

AetherLink bridges your Hytale server with Discord, allowing seamless chat synchronization and event notifications.

---

## Feature Overview

| Feature | Status |
|---------|--------|
| **Discord → Hytale Chat** | ✅ Working |
| **Hytale → Discord Chat** | ✅ Working |
| **Player Join/Leave Notifications** | ✅ Working |
| **Multi-Channel Support** | ✅ Working |
| **Per-Channel Read-Only Mode** | ✅ Working |
| **Cross-Channel Sync** | ✅ Working |
| **Discord Slowmode (Spam Control)** | ✅ Working |
| **Hytale Duplicate Aggregation** | ✅ Working |
| **Auto-Retry Discord Connection** | ✅ Working |
| **Bot Presence Updates** | ✅ Working |
| **Channel Topic Updates** | ✅ Working |
| **In-Game Commands** (`/aetherlink`) | ✅ Working |
| **Configurable Messages** | ✅ Working |
| **Discord Account Linking** | ⚠️ Not Implemented |
| **Death Messages** | ⚠️ Not Implemented|
| **Zone/Advancement Unlocks** | ⚠️ Not Implemented  |
| **Max Players Display** | ⚠️ Not Implemented |
